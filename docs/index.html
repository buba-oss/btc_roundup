Privacy Policy for BTC Round Up

Last Updated: February 21, 2026

1. Information We Collect

We collect information to provide our Bitcoin round-up investment services:

  • Account Information: Email address, username, password
  • Financial Data: Bank account connections (via secure APIs), transaction data for round-up calculations, Bitcoin wallet addresses
  • Device Information: Device ID, OS version, app version (for security and troubleshooting)
  • Usage Data: App interaction data, crash reports (via Firebase Analytics)

2. How We Use Your Information

  • To calculate purchase round-ups and execute Bitcoin investments

  • To maintain account security and prevent fraud
  • To improve app functionality and user experience

3. Data Sharing

We share data only with:

  • Cryptocurrency Exchanges: For executing Bitcoin purchases (e.g., Coinbase, Kraken)

4. Data Security

We implement industry-standard encryption (TLS 1.3 for transmission, AES-256 for storage). We do not store full bank account numbers or private keys.
